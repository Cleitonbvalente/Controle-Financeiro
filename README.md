# Controle Financeiro

Sistema web para controle de receitas e despesas pessoais.

## Funcionalidades

- Cadastro de receitas
- Cadastro de despesas
- Listagem de receitas
- Listagem de despesas
- Cálculo de saldo
- Filtro por mês
- Persistência com LocalStorage

## Tecnologias

- HTML
- CSS
- JavaScript
- LocalStorage

## Conceitos utilizados

- Classes
- Map
- Arrow Functions
- Reduce
- Filter
- Sort
- Find
- Spread Operator
- Template Literals
- Closure
- IIFE
- Import / Export
